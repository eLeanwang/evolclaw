# EvolClaw 轻量化重构项目

## 项目概述

**目标**：简化 EvolClaw 架构，从 6 层 4 表简化为 4 层 2 表，用独立日志文件替代数据库监控表。

**预期成果**：
- 减少 50% 数据库表
- 减少 60% 新增代码量
- 减少 40% 工作量
- 保持核心功能完整

## 风险管理

### 已识别风险
1. **性能风险**：数据库查询可能影响性能
   - 缓解：添加索引，性能测试

2. **兼容性风险**：日志格式变化可能影响现有工具
   - 缓解：向后兼容设计

3. **测试风险**：监控组件从未使用，可能有未知 bug
   - 缓解：全面测试，分阶段实施

### 回滚方案
- 保留 git tag
- 数据库表向后兼容
- 分阶段实施，每阶段测试
