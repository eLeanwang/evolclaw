## 测试覆盖的功能点

### 核心功能
1. ✅ 消息缓存机制
   - 内存缓存，按 sessionId 组织
   - 缓存上限 100 条消息
   - 自动清理过期消息（1小时）
   - FIFO顺序保持

2. ✅ 项目切换时的消息路由
   - 活跃项目：直接发送
   - 非活跃项目：缓存消息
   - 使用 AsyncLocalStorage 传递上下文
   - 嵌套上下文支持

3. ✅ Interrupt 机制优化
   - 同项目：触发中断
   - 不同项目：不触发中断
   - 路径格式容错
   - 回调错误处理

4. ✅ 文件处理
   - 文件存在：正常发送
   - 文件不存在：跳过发送，提示用户
   - 多文件标记支持
   - 绝对/相对路径支持

5. ✅ 多聊天/多channel独立性
   - 每个聊天的缓存独立
   - 不同channel（Feishu/AUN）独立
   - 互不干扰
